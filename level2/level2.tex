\subsubsection{パラメータの模索方法}
パラメーターは以下のようにして、実際に値を入れてみて、結果を見ながら決定した。\\
誤差のみを見るのではなく、キチンと収束するかどうかも調べながら模索した。\\
\begin{itemize}
\item alpha n :::回数,誤差
 \item alpha 0.1 :::394,0.0000997313
 \item alpha 0.5 :::17,0.0000998573
 \item alpha 0.8 :::87,0.0000938422
 \item alpha 0.89 :::48,0.0000934068
 \item alpha 0.90 :::44,0.0000820184
 \item alpha 0.94 :::32,0.0000489520
\end{itemize}
このような結果になったが、alpha=0.89以上の値は収束する回数が増えていくため、alpha=0.89がベストだと考えた。\\
